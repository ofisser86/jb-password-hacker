Re-write using lambda functions

Re-write the following piece of code using lambda functions:

a = 1
b = 2
c = 3

def func(a, b, c):
    return (a + b)*c

result = func(a, b, c)

Assign the resulting function to the variable result.

Since lambda functions are anonymous, they are rarely assigned to variables. Ignore the corresponding code style error. The assignment is here for test purposes.



Memory limit: 256 MB
Time limit: 15 seconds

Caution

You may see errors in your code or in execution results because of missing context. Don't mind those; simply write your solution and click Check.
